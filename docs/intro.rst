Installation
------------

Django Site Utils is available on PYPI to be installed with either easy_install
or pip::

   pip install django-site-utils
   
   easy_install django-site-utils

Configuration
-------------

Add site_utils to your Django project's INSTALLED_APPS::

    INSTALLED_APPS = (
       ...
       'site_utils',
       ...
    )
